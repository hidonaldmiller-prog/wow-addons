# wow-addons

时光服的一些插件进行了改良，变得好用。  PS：这些插件非本人开发，只是在原作者的基础上进行了一定的改动。

The add-ons for Time Classic server have been fully optimized for better practical performance.

cell插件的BUF监控，我额外添加了食物与合剂的监控与播报

MRT的插件之前存在BOSS战结束后，技能CD监控不刷新的问题，我把它修复了

DCT插件的显示框，我额外添加了显示的类型，更加直观的看到这个框显示的东西，另外增加了鼠标拖动显示框的功能，更加方便的移动到自己想要的位置
